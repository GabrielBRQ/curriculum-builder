# 📄 Curriculum Builder

> Uma aplicação web interativa para criação e formatação de currículos profissionais em tempo real.

🔗 **[Acesse a aplicação rodando aqui](https://curriculum-builder-psi.vercel.app/)**

## 💡 Sobre o Projeto
O Curriculum Builder foi desenvolvido para simplificar o processo de criação de currículos. Com uma interface intuitiva, o usuário pode preencher suas informações pessoais, experiências e formação acadêmica, visualizando o resultado final instantaneamente. Este projeto foi construído com o objetivo de aplicar conceitos avançados de componentização e gerenciamento de estado no Front-end.

## ✨ Funcionalidades
- **Live Preview:** Veja as alterações no layout do currículo em tempo real enquanto digita os dados.
- **Seções Dinâmicas:** Adicione, edite ou remova experiências profissionais e formações acadêmicas de forma fluida.
- **Design Responsivo:** Interface adaptada para oferecer uma boa experiência em diferentes tamanhos de tela.
